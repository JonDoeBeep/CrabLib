Detailed Description

template<typename T, std::size_t Capacity>
class crab::StaticRingBuffer< T, Capacity >

Lock-free SPSC ring buffer with fixed capacity.

Provides wait-free push/pop operations for real-time systems. Uses acquire/release memory ordering for proper synchronization.

Template Parameters
TElement type (should be trivially copyable for best performance)
CapacityMaximum number of elements (actual usable = Capacity - 1)
// Producer thread
crab::StaticRingBuffer<SensorData, 64> buffer;
buffer.try_push(SensorData{...});
// Consumer thread
if (auto data = buffer.try_pop(); data.is_some()) {
process(data.unwrap());
}
