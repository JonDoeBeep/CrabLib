Detailed Description

template<typename T, std::size_t Capacity>
class crab::StaticVector< T, Capacity >

Fixed-capacity vector with no heap allocation.

Stores elements in-place using aligned storage. Suitable for real-time and embedded systems.

Template Parameters
TElement type
CapacityMaximum number of elements
