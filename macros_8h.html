Detailed Description

CrabLib assertion and utility macros.

Provides debug-mode assertions and helper macros for safe error handling. Designed for -fno-exceptions environments and bare-metal embedded.

Customization

  • CRAB_CUSTOM_PANIC: Define before including to use custom panic handler
  • CRAB_CACHE_LINE_SIZE: Define to override default cache line (32 or 64)
