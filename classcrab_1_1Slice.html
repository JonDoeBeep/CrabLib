Detailed Description

template<typename T>
class crab::Slice< T >

Bounds-checked view into contiguous memory.

Like std::span but for C++17, with mandatory bounds checking. Does NOT own the data. Caller must ensure the underlying buffer outlives the Slice.

Template Parameters
TElement type (can be const for read-only slices)
std::vector<uint8_t> buffer(100);
crab::Slice<uint8_t> slice(buffer); // From lvalue container
auto elem = slice.get(5); // Returns Result<ref, OutOfBounds>
if (elem) {
*elem.unwrap() = 42;
}
// This won't compile, prevents dangling:
// crab::Slice<uint8_t> bad(std::vector<uint8_t>(100)); // DELETED
