Detailed Description

template<typename T, typename E>
class crab::Result< T, E >

Result type for operations that can fail.

Models Rust's Result<T, E>. Either contains a success value (Ok) or an error value (Err). Uses std::variant internally for storage.

Template Parameters
TSuccess value type (use Unit for void operations)

Result<int, OutOfBounds> get_element(size_t i) {
if (i >= size) return Err(OutOfBounds{i, size});
return Ok(data[i]);
}
// Usage:
auto result = get_element(5);
if (result.is_ok()) {
int val = result.unwrap();
}
// Or with CRAB_TRY:
int val = CRAB_TRY(get_element(5));
