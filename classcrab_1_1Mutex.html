Detailed Description

template<typename T>
class crab::Mutex< T >

Data-owning mutex (Rust-style).

The mutex owns the data it protects. The only way to access the data is through the Guard returned by lock() or try_lock().

Template Parameters
TProtected data type
crab::Mutex<std::vector<int>> data(std::vector<int>{1, 2, 3});
// Data can only be accessed while holding the lock
{
auto guard = data.lock();
guard->push_back(4);
(*guard)[0] = 10;
} // Lock released here
// try_lock for non-blocking
if (auto guard = data.try_lock(); guard.is_some()) {
guard.unwrap()->clear();
}
