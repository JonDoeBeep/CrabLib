Detailed Description

template<typename T, typename LockType = StdMutexLock>
class crab::Mutex< T, LockType >

Data-owning mutex (Rust-style).

The mutex owns the data it protects. The only way to access the data is through the Guard returned by lock() or try_lock().

Template Parameters
TProtected data type
LockTypeLock implementation (default: StdMutexLock)
